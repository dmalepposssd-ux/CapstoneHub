# هيكل السلايدات

## 1. المشكلة

تشتت متابعة مشاريع التخرج بين الطلاب والمشرفين والإدارة، وصعوبة معرفة حالة كل مشروع أو اكتشاف التعثر مبكراً.

## 2. الحل

CapstoneHub منصة محلية تجمع دورة حياة مشروع التخرج من الفكرة حتى المتابعة والتقييم، مع مساعد ذكي يدعم القرار.

## 3. أدوار النظام

طالب، مشرف، إدارة. كل دور يملك لوحة وصلاحيات ومسارات مختلفة.

## 4. المعمارية

React Frontend، Express Backend، FastAPI AI Service، PostgreSQL + pgvector، وتشغيل Docker Compose محلي.

## 5. قاعدة البيانات

Users, Students, Supervisors, Projects, Milestones, Submissions, Messages, Notifications, Blueprints, AI Documents.

## 6. الذكاء الاصطناعي

فحص اكتمال الفكرة، أسئلة متابعة، RAG، تشابه المشاريع السابقة، اقتراح المشرفين، Blueprint، تحليل الأطروحة، ولوحة مخاطر.

## 7. سيناريو الطالب

تسجيل الدخول، الاستبيان، إنشاء فكرة، توليد Blueprint، رسم Mermaid، تصدير صورة، مشاركة المخطط، رفع فصل وتحليله.

## 8. سيناريو المشرف

مراجعة طلبات المشاريع، قبول/تعديل/رفض، متابعة الفصول والمخططات، تعليق على المخطط، تقييم Blueprint، إضافة مراحل.

## 9. سيناريو الإدارة

الإحصاءات، المستخدمون، تفعيل الحسابات، التقويم، سعات المشرفين، المشاكل التقنية، المخاطر، أرشفة المشاريع.

## 10. النتائج

نسخة محلية مستقرة، بيانات Demo جاهزة، خدمات healthy، توثيق كامل، وسيناريو عرض قابل للتنفيذ.

## 11. القيود

الذكاء إرشادي، جودة النتائج تعتمد على وصف الطالب والبيانات، ولا يوجد نشر Online في نسخة العرض.

## 12. التطوير المستقبلي

تقييمات كمية أوسع، تحسين فهرسة الوثائق، لوحات تقارير أعمق، اختبارات تلقائية شاملة، وتكامل لاحق مع أنظمة الجامعة بعد الموافقة.
