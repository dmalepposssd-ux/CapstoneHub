# أمثلة مختصرة للمساعد أثناء المناقشة

استخدم هذه الأمثلة إذا أردت عرض سلوك المساعد بسرعة.

## فكرة ناقصة

```text
بدي نظام ذكي للجامعة
```

المتوقع: أسئلة متابعة وثقة منخفضة.

## فكرة كاملة

```text
منصة لإدارة مشاريع التخرج بين الطلاب والمشرفين والإدارة. الطالب يرسل مقترحاً ويرفع الفصول، المشرف يراجع ويعلق، والإدارة ترى المخاطر. التقنيات React وNode.js وPostgreSQL وFastAPI. الذكاء يفحص التشابه ويقترح مشرفاً ويولد Blueprint.
```

المتوقع: Blueprint وثقة مرتفعة نسبياً وخطة تنفيذ.

## فكرة مشابهة للأرشيف

```text
تطبيق تتبع باصات الجامعة مع مسارات وتنبيهات ومواعيد وصول.
```

المتوقع: أدلة من مشاريع مؤرشفة ونسبة تشابه وتوصيات للتمييز.
