# سيناريو عرض CapstoneHub

## قبل البدء

شغل المشروع:

```powershell
docker compose up -d --build
docker compose ps
```

تأكد أن الخدمات `healthy` وافتح:

```text
http://localhost:5173
```

## 1. دخول الطالب

استخدم:

```text
student1@capstonehub.local
Password123!
```

اعرض:

- لوحة الطالب وجاهزية المشروع.
- الاستبيانات: الاستبيان يظهر للإجابة مرة واحدة، وبعد الإجابة يصبح للعرض فقط.
- طلب المشروع وبنك الأفكار والمشاريع المشابهة.
- المساعد الذكي بفكرة ناقصة ثم فكرة كاملة.
- توليد Blueprint ورؤية الجداول والعلاقات.
- محرر المخططات: Mermaid، إضافة نص، مربع، سهم، ثم تصدير PNG.
- مشاركة المخطط مع المشرف.
- رفع فصل أو فتح الفصل الموجود وتحليل الملف بالذكاء.
- الضغط على تنبيه والتأكد أنه ينقل للقسم المناسب.
- فتح المحادثة مع المشرف.

## 2. دخول المشرف

استخدم:

```text
sara@capstonehub.local
Password123!
```

اعرض:

- لوحة المشرف وعدد الطلاب.
- المقترحات الجديدة: قبول، طلب تعديل، أو رفض مع ملاحظة.
- متابعة المشاريع: مخططات الطلاب، صورة المخطط، كود Mermaid، الفصول، والملاحظات.
- فتح صورة المخطط بشكل كامل.
- إرسال تعليق للطالب من المحادثة.
- تقييم Blueprint وحفظ ملاحظات المشرف.
- إضافة مرحلة جديدة وتحديث حالتها.
- فتح تحليل المساعد للملفات.

## 3. دخول الإدارة

استخدم:

```text
admin@capstonehub.local
Password123!
```

اعرض:

- لوحة الإحصاءات.
- إدارة الطلاب والمشرفين وتأكيد الملفات.
- إنشاء حساب جديد محلي بكلمة سر عشوائية.
- لوحة المخاطر والمشاريع المتعثرة.
- المشاكل التقنية مع لقطة شاشة.
- التقويم الأكاديمي وسعات المشرفين.
- إدارة التنبيهات والاستبيانات.
- أرشفة مشروع منته كمرجع للمشاريع القادمة.

## نقاط قوية لذكرها

- النظام متعدد الأدوار وليس شاشة واحدة عامة.
- كل حساب يرى ما يخصه حسب الصلاحيات.
- بيانات Demo واقعية: مشاريع مؤرشفة، مشروع قيد المراجعة، مشروع مقبول، مشروع بحاجة تعديل، ملفات وتحليلات.
- المساعد يعطي أسئلة متابعة ولا يرفع الثقة عندما تكون الفكرة غامضة.
- التصدير كصورة يحول النصوص إلى SVG text قبل PNG حتى لا تضيع النصوص.
- لا يوجد Deploy Online؛ العرض محلي بالكامل.

## أسئلة متوقعة

- كيف منعتم الطالب من رؤية مشاريع غيره؟
- ما الفرق بين RAG وفحص التشابه البسيط؟
- لماذا استخدمتم pgvector؟
- كيف تقيمون جودة Blueprint؟
- كيف تعرف الإدارة أن مشروعاً قد يتعثر؟
- ما حدود المساعد الذكي؟
